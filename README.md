# Culturalverse

Where Culture Becomes Place.

An immersive culture-world layer where living traditions are explored through place, protocol, story, symbol, sound, and relationship.

## Technology Direction

### Initial stack

- HTML
- CSS
- JavaScript
- Three.js
- GSAP animations
- localStorage for lightweight state
- shared ecosystem links

### Future stack

- Supabase profile sync
- Supabase saved reflections
- user favorites
- saved world paths
- soundscape preferences
- PWA / offline cultural packs
- optional WebXR world spaces
- integration with Wayfinder Passport
- future XRPL-ready learning records through LKP

## Guiding Principle

Knowledge is not only data.

It is carried by people.
It is rooted in place.
It is shaped by language.
It is protected by protocol.
It is renewed through practice.
It is inherited with responsibility.

Culturalverse exists to help learners approach that responsibility with respect.
